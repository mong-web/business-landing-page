A high-converting, fully responsive business landing page built with pure HTML5, CSS3, and JavaScript. Designed for agencies, startups, and freelancers to showcase services, build trust, and capture leads.

🔗 [Live Demo](https://super-dodol-d0549a.netlify.app/) | 📂 [View Source](https://github.com/mong-web/business-landing-page)

---

- ✅ **Fully Responsive** – Flawless on Mobile, Tablet & Desktop
-  **Modern UI/UX** – CSS Grid, Flexbox & Glassmorphism Cards
- ⚡ **Performance Optimized** – Fast loading, clean semantic code
-  **Mobile Navigation** – Smooth hamburger menu with animations
- 📬 **Working Contact Form** – Integrated with Formspree (No backend needed)
- 🌊 **Scroll Animations** – Intersection Observer powered reveal effects
- 🎯 **SEO Friendly** – Proper heading hierarchy & meta structure
- 🛠️ **Easy Customization** – CSS Variables for quick color/font changes

1. **Clone or Download** the repository
2. Open `index.html` in any code editor
3. **Customize Colors:** Edit the `:root` CSS variables at the top of the file
4. **Contact Form:** Replace `action="https://formspree.io/f/YOUR_ID"` with your Formspree endpoint
5. Deploy to Netlify/Vercel or host on any static server
